Alternatives to Finder for file management on macOS, mostly dual-pane oriented.

- **[ForkLift 4](https://binarynights.com/)** — dual-pane file manager with built-in FTP, SFTP, S3, and WebDAV support
- **[Marta](https://marta.sh/)** — native macOS dual-pane file manager with a plugin system and keyboard-centric workflow
- **[muCommander](https://www.mucommander.com/)** — open-source, cross-platform dual-pane file manager built on Java
- **[Commander One](https://mac.eltima.com/file-manager.html)** — dual-pane Finder replacement with FTP client and archive support
