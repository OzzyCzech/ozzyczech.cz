This is my personal knowledge base — a public collection of notes I've gathered over time.
It works as an [LLM Wiki](/ai/guides/llm-wiki/): I curate the sources and direction,
an LLM agent helps organize, summarize, and keep things consistent.
It's [open source](https://github.com/OzzyCzech/ozzyczech.cz) and maintained by [Roman Ožana](https://ozana.cz).

### 📝 What's it about?

Notes are mostly about web development, AI, and other **things I'm interested in**.
Things I've **googled a lot** and want to remember, as well as things
I find interesting and worth sharing.

### 🚀 Can I contribute?

Yes. The source code is on [GitHub](https://github.com/OzzyCzech/ozzyczech.cz).
If you find any bugs or outdated information, please create a [pull request](https://github.com/OzzyCzech/ozzyczech.cz/pulls).

If you want to support me, you can [buy me a coffee](https://buymeacoffee.com/ozzyczech).

```shell
git clone git@github.com:OzzyCzech/ozzyczech.cz.git
```

```shell
npm install
npm run dev
```
