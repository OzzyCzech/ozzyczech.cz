### Linters & Formatters

- [oxc](https://oxc.rs/) — Fast, modern, and secure JavaScript linter and formatter.
- [Biome.js](https://biomejs.dev/) — Tool-chain for web development.


### Testing Frameworks

- [Vitest](https://vitest.dev/) — Modern, fast, and reliable testing framework for JavaScript.
- [Jest](https://jestjs.io/) — Delightful JavaScript testing framework focusing on simplicity.
- [Mocha](https://github.com/mochajs/mocha) — Simple, flexible, fun test framework for Node.js & browser.
- [AVA](https://github.com/avajs/ava) — Futuristic JavaScript test runner.
- [Jasmine](https://github.com/jasmine/jasmine) — DOM-less, simple JavaScript testing framework.


### End-to-End and Automation Testing

- [Puppeteer](https://pptr.dev/) — Headless Chrome Node.js API from the official Google Chrome team.
- [Playwright](https://github.com/microsoft/playwright) — Framework for web testing and automation.
- [Cypress](https://www.cypress.io/) — Fast, easy, and reliable testing for anything running in a browser.


### Additional Libraries

- [Polly.js](https://netflix.github.io/pollyjs) — Record, replay, and stub HTTP interactions.
