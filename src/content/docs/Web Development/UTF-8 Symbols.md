## Typography

| Symbol | Description         | HTML Entity | Mac Shortcut | 
| ------ | ------------------- | ----------- | --------- |
| `❤`   | Heavy black heart   | `&#10084;`  | |
| `©`   | Copyright           | `&copy;`    | |
| `®`   | Registered trademark| `&reg;`     | Opstions + Shift + r |
| `°`    | Degree sign         | `&deg;`     | |
| `•`    | Bullet              | `&bull;`    | Options + Shift + 2 |
| `*`    | Asterisk            | `&ast;`     | Options + 8 |
| `…`    | Horizontal Ellipsis | `&hellip;`  | Options + Shift + ů |
