### Minecraft Commands

##### Setup game mode

```text
gamemode creative @a
gamemode survival @a
```

##### Kill all entities

```text
kill @e[type=warden]
```

##### Give items

```text
give @a minecraft:apple 10
give @a minecraft:netherite_axe
```

All items are listed on [Minecraft Item ID List](https://minecraftitemids.com/).

### Show coordinates

```text
/gamerule showcoordinates true
```
