- **[Hugging Face](https://huggingface.co/)**
  Collaborative platform for sharing and discovering AI models, datasets, and applications.
- **[Ollama](https://ollama.com/)**
  Run large language models locally on your machine.
- **[Replicate](https://replicate.com/)**
  Run open-source AI models in the cloud via simple API.
- **[OpenRouter](https://openrouter.ai/)**
  Unified API for accessing models from OpenAI, Anthropic, Google, Meta and others.
- **[LM Studio](https://lmstudio.ai/)**
  Desktop app for running local LLMs with a simple GUI.
