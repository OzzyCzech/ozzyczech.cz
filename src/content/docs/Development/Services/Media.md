Media server software for streaming personal video and audio libraries to any device.

## Media servers

- **[Jellyfin](https://jellyfin.org)** — free and open-source media server with no licensing fees or account requirements
- **[Plex](https://plex.tv)** — media server with a polished client ecosystem across TVs, mobile, and web
- **[Emby](https://emby.media)** — self-hosted media server with live TV support and parental controls
